# Changelog

## otto 0.1.0 (baseline)

Initial release of the Otto CLI, exposing the `otto-afk` (plan/PRD loop) and
`otto-ghafk` (GitHub-issue loop) bin entries, depending on
`@phamvuhoang/otto-core`. The CLI is versioned independently of `otto-core`; its
entries always appear under their own `otto` heading.

release-please prepends new release sections above this baseline. See
`RELEASING.md` for the release process and commit conventions.
